Generalized UMAP Analysis Process
  1. Perform UMAP Clustering using many different values for n_neighbors
  2. Make subjective judgement for n_neighbors that makes sense, if any
  3. Perform clustering using algorithm appropriate to structure
  4. T-test clusters to investigate what features are driving UMAP structure
  5. Permute labels and t-test repeatedly to decide if significance is likely by chance
